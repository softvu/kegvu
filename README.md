# kegvu

## Project setup
```
nvm use 8
npm install
```

- Add UNTAPPD.client_secret, UNTAPPD.client_id, and UNTAPPD.beerExample to src/config.js

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Lints and fixes files
```
npm run lint
```
